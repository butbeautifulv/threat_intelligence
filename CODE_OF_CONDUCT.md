# Code of conduct

We want a respectful, professional environment for everyone working on or using this repository.

## Our standards

- **Be respectful** — disagree on ideas, not people; assume good intent.
- **Be inclusive** — welcome newcomers; avoid demeaning or discriminatory language.
- **Be constructive** — clear bug reports and reviews help the project; pile-ons do not.

## Unacceptable behaviour

Harassment, threats, publishing others’ private information, sustained disruption of discussions, or other conduct that would be inappropriate in a professional setting are not tolerated.

## Reporting

If you experience or witness behaviour that violates this code, please contact the repository maintainers (for example via GitHub issues if appropriate, or a private channel they provide). Reports will be reviewed promptly and confidentially to the extent possible.
